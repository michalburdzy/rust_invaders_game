# Simple Invaders game

Created thanks to Nathan Stocks's great [Ultimate Rust Crash Course](https://www.udemy.com/course/ultimate-rust-crash-course)

## How to play
- Run `invaders` file

- Use left / right arrow keys to steer

- use Space or Enter key to shoot

- if you kill all invaders, you win

- if invaders touch the ground, you lose

- type `q` to exit
